# OtoBattle

音楽 × サバイバルゲーム

**Play: https://lutelute.github.io/otobattle/**

四方から迫る敵の音符を見て、正しい音を鳴らして撃破するブラウザゲーム。

## 入力方法
- マイク（楽器演奏・歌声）
- MIDIキーボード
- タッチピアノ（画面下部）
- PCキーボード: C D E F G A B（Shift=シャープ）

## 対応楽器
ピアノ / バイオリン / ビオラ / チェロ / ギター / フルート / 声

## 技術スタック
React + TypeScript + Vite + Tailwind CSS + Canvas + Web Audio API
